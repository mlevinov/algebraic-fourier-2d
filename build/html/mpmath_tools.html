mpmath_tools.elementwise_norm_matrix(mp_arr1, mp_arr2)[source]¶

Calculates \(\lvert \text{mp_arr1[i,j]-mp_arr2[i,j]} \rvert\) for each \(1\leq i\leq m\) and \(1\leq j\leq n\) where mp_arr1, mp_arr2 are of order \(m\times n\)

Parameters:

  • mp_arr1 – (mpmath.matrix)

  • mp_arr2 – (mpmath.matrix)

Returns:

a matrix where each element is the absolut value of the difference between mp_arr1[i,j] and mp_arr2[i,j]

Return type:

mpmath.matrix
