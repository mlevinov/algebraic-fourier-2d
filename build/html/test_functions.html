test_functions module¶

class test_functions.TestFunctions(dps=25, func_type=1)[source]¶

Bases: object

Class TestFunctions represents three functions with two variables which are the test subjects of our routine. The class contains method which provides more data for each of the three test functions, such as its discontinuity curves and ect. dps value holds the precision in digits and its defaults is 25 (DEFAULT_DPS)
